# 贡献指南

感谢你对本项目的兴趣！

## 如何贡献

1. Fork 这个仓库
2. 创建新分支 (git checkout -b feature/功能名称)
3. 提交更改 (git commit -m '添加新功能')
4. 推送分支 (git push origin feature/功能名称)
5. 创建 Pull Request

## 贡献 ideas

- 文档改进
- 错误修复
- 新功能建议
- 代码优化

## 指南

- 遵循现有代码风格
- 提交前测试脚本
- 保持提交信息清晰
